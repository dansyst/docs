## Windows 引导项问题

### 1. Windows引导修复软件

#### 制作U盘教程 

用另外好的电脑和空U盘下载制作U盘工具

先退出电脑上所有杀毒（360 电脑管家等）然后按教程操作

[软件下载网站](zyz.lanzn.com/iHxWe2chwbhc)

[相对路径：在当前域名后添加括号中的内容](ldn/)

这个是制作U盘的工具，浏览器下载

- 把这个文件下载到电脑里（不要下载到U盘）

- 重要提示U盘必须是空的  因为制作过程会格式化U盘

- 然后打开制作U盘工具选一键制作，再选仅制作启动U盘

制作完成后会有完成弹窗
